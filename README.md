# ProposalPilot — AI Proposal Generator for Freelancers

**Describe your project in plain English → get a professional proposal in 90 seconds.** Smart pricing, PDF export, 6 project templates.

🔗 **Live:** [https://atalovesyou.github.io/proposalpilot/](https://atalovesyou.github.io/proposalpilot/)

## Features

- **NLP Proposal Generation** — Describe project in natural language → structured proposal
- **6 Project Templates** — Web dev, design, marketing, video, writing, consulting
- **Smart Toggles** — Rush fee, revision rounds, deposit %, kill fee, NDA
- **AI Pricing Tool** — Market-rate suggestions (Low/Market/Premium)
- **PDF Export** — Professional formatting with jsPDF
- **Comparison Table** — vs Proposify, Better Proposals, PandaDoc, Google Docs
- **100% Client-Side** — No data leaves your browser
